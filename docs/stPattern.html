Signaling Patterns and Velocities for Spatial Transcriptomics

Python GPU-accelerated Adapted from the SecAct R tutorial using the spatial-gpu package

Overview

This tutorial demonstrates secreted protein activity inference on a liver cancer (HCC) Visium dataset. Secreted proteins are key mediators of intercellular communication, and their activity can be inferred from downstream target gene expression using ridge regression.

The workflow covers: (1) activity inference for ~500 secreted proteins, (2) signaling pattern discovery via non-negative matrix factorization (NMF) to identify spatially coherent signaling programs, (3) signaling velocity visualization to show the direction and magnitude of signaling flow, (4) cell type deconvolution, and (5) hallmark gene set scoring.

The spatial-gpu package integrates the SecAct algorithm (Nature Communications 16, 1559, 2025) with the SpaCET deconvolution framework. All results are stored in the AnnData object under adata.uns['spacet'].

1. Load ST Data

Load the Visium hepatocellular carcinoma (HCC) dataset from the SecAct package. This dataset is provided in 10X Space Ranger format. 

import spatialgpu.deconvolution as spacet

# Load Visium HCC data (10X format from SecAct package)
adata = spacet.create_spacet_object_10x("data/Visium_HCC")

# Quality control: filter spots with < 1000 genes
adata = spacet.quality_control(adata, min_genes=1000)

print(adata)

2. Infer Secreted Protein Activity

Infer the activity of >1,000 secreted proteins using ridge regression on downstream target gene expression. The number of proteins with significant results depends on the gene coverage in the dataset. The scale_factor parameter normalizes UMI counts to a common library size before activity inference. 

# Infer secreted protein activity (~10 min)
adata = spacet.secact_inference(adata, scale_factor=1e5)

# Access z-score activity matrix (proteins x spots)
zscores = adata.uns['spacet']['SecAct_output']['SecretedProteinActivity']['zscore']
print(f"Inferred activity for {zscores.shape[0]} proteins across {zscores.shape[1]} spots")
Output
Inferred activity for 489 proteins across 3404 spots
# Visualize activity of selected secreted proteins
spacet.visualize_spatial_feature(
    adata,
    spatial_type="SecretedProteinActivity",
    spatial_features=["HDGF", "MYDGF"],
    ncols=2,
)
Spatial activity of HDGF and MYDGF in the HCC Visium dataset. Color scale represents z-score normalized activity: blue (low) to red (high).
Note: SecAct activity z-scores represent the deviation of inferred activity from a permutation-based null distribution. Positive z-scores indicate elevated secreted protein activity relative to the background.

3. Estimate Signaling Patterns

Discover spatially coherent signaling programs using non-negative matrix factorization (NMF) on the activity z-score matrix. The parameter k specifies the number of signaling patterns to extract. 

# Estimate signaling patterns via NMF (~3 min)
adata = spacet.secact_signaling_patterns(adata, k=3)

# Visualize the discovered signaling patterns
spacet.visualize_spatial_feature(
    adata,
    spatial_type="SignalingPattern",
    spatial_features=["1", "2", "3"],
    ncols=3,
)
Three signaling patterns discovered by NMF. Each pattern represents a spatially coherent secreted protein signaling program. Color intensity reflects the pattern loading per spot.

Identify pattern-defining genes

For each signaling pattern, identify the top secreted proteins with the highest NMF loadings: 

# Get secreted proteins dominated by pattern 3
pattern3_genes = spacet.secact_pattern_genes(adata, n=3)
print(pattern3_genes.head())

4. Calculate Signaling Velocity

Signaling velocity visualizes the direction and magnitude of secreted protein signaling flow across the tissue. SPARC, associated with epithelial-mesenchymal transition, is used as an example. The velocity can be displayed as a contour map (smoothed flow field) or as individual spot-level arrows. 

# Calculate signaling velocity for SPARC
velocity = spacet.secact_signaling_velocity(adata, gene="SPARC")

# velocity is a dict with 'arrows' (DataFrame) and 'points' (DataFrame)
print(f"Velocity arrows: {len(velocity['arrows'])} spots with non-zero flow")

# Plot as contour map (smoothed signaling flow)
import matplotlib.pyplot as plt
import numpy as np

fig, ax = plt.subplots(figsize=(8, 7))
arrows = velocity['arrows']
coords_x = adata.obs['coordinate_x_um'].values
coords_y = adata.obs['coordinate_y_um'].values
ax.scatter(coords_x, coords_y, s=0.5, c='lightgrey', alpha=0.3, edgecolors='none')
if len(arrows) > 0:
    ax.quiver(
        arrows['x_start'].values, arrows['y_start'].values,
        arrows['x_change'].values, arrows['y_change'].values,
        arrows['vec_len'].values, cmap='coolwarm',
        width=0.003, alpha=0.7,
    )
ax.set_title('SPARC Signaling Velocity')
ax.set_aspect('equal')
ax.invert_yaxis()
ax.axis('off')
Signaling velocity of SPARC in the HCC tissue. Arrows indicate the direction of signal propagation from source (high expression) to sink (high activity) cells. SPARC is associated with the epithelial-mesenchymal transition process.
Note: Signaling velocity captures the spatial flow of secreted protein signaling. Arrows pointing from high-expression to high-activity regions suggest paracrine signaling, while convergent arrows may indicate autocrine signaling hotspots.

5. Deconvolve ST Data

Apply SpaCET deconvolution to estimate cell-type fractions in the HCC Visium data. The cancer type "LIHC" (liver hepatocellular carcinoma) uses liver-specific CNA and expression signatures. 

# Run deconvolution for liver cancer
adata = spacet.deconvolution(adata, cancer_type="LIHC", n_jobs=8)

# View selected cell type fractions
prop_mat = adata.uns['spacet']['deconvolution']['propMat']
print(prop_mat.loc[
    ["Malignant", "CAF", "Endothelial", "Macrophage", "Hepatocyte"],
    prop_mat.columns[:5]
])

6. Calculate Hallmark Score

Score each spot for MSigDB Hallmark gene sets to characterize the functional state of the tissue. Gene set scores are computed using the UCell algorithm (rank-based), which is robust to library size differences across spots. 

# Calculate Hallmark gene set scores
adata = spacet.gene_set_score(adata, gene_sets="Hallmark")

# Visualize EMT hallmark score
spacet.visualize_spatial_feature(
    adata,
    spatial_type="GeneSetScore",
    spatial_features=["HALLMARK_EPITHELIAL_MESENCHYMAL_TRANSITION"],
)
Spatial distribution of the Hallmark Epithelial-Mesenchymal Transition (EMT) gene set score. High scores (red) indicate spots with elevated EMT-associated gene expression, often corresponding to tumor-stroma boundary regions.
Note: Hallmark gene set scores can be combined with deconvolution results to characterize the functional state of specific cell types. For example, correlating EMT scores with CAF fractions can reveal fibroblast-driven EMT programs at the tumor-stroma interface.
