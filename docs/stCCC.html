Cell-Cell Communication for Single-Cell Resolution ST

Python GPU-accelerated Adapted from the SecAct R tutorial using the spatial-gpu package

Overview

This tutorial demonstrates inference of cell-cell communication (CCC) mediated by secreted proteins using single-cell resolution spatial transcriptomics data. We use a CosMx liver cancer (LIHC) dataset containing 460,441 cells measured across 1,000 genes.

Unlike spot-level Visium data where each measurement captures 1–10 cells, single-cell resolution platforms like CosMx, Xenium, and MERSCOPE resolve individual cells. This enables direct inference of sender-receiver relationships: which cell types secrete specific proteins and which cell types respond to them.

The workflow covers: (1) activity inference at single-cell resolution, (2) spatial cell-cell communication inference using neighborhood analysis, and (3) visualization of CCC results including heatmaps, circle plots, dot plots, and signaling velocity.

1. Load ST Data

Load the CosMx LIHC dataset from a pre-processed H5AD file. This dataset contains 460,441 cells with 1,000 measured genes and pre-annotated cell type labels. 

import spatialgpu.deconvolution as spacet
import anndata as ad

# Load CosMx LIHC data
adata = ad.read_h5ad("data/LIHC_CosMx/LIHC_CosMx.h5ad")
print(adata)

2. Infer Secreted Protein Activity

The secact_inference() function outputs four items: beta (regression coefficients), se (standard error), zscore (beta/se), and pvalue. The scale_factor is set to 1,000 for CosMx data. 

# Infer secreted protein activity at single-cell resolution (~30 min)
adata = spacet.secact_inference(
    adata,
    scale_factor=1000,
)

# Check inferred activities
zscores = adata.uns['spacet']['SecAct_output']['SecretedProteinActivity']['zscore']
print(f"Inferred activity for {zscores.shape[0]} proteins across {zscores.shape[1]} cells")
Output
Inferred activity for 312 proteins across 443515 cells

3. Infer Cell-Cell Communication

First, filter secreted proteins using Spearman correlation of secreted protein signaling activity and neighbor cells’ sum expression. P values are adjusted by Benjamini-Hochberg (BH) method as false discovery rate (FDR). Cutoffs: r > 0.05 and FDR < 0.01.

Second, for each secreted protein, identify neighboring pairs of cells where distance < 20 μm. The communication score is defined as the ratio of communicating pairs to total neighboring pairs. A pair is considered communicating when a sender cell expresses the protein and a receiver cell has high inferred activity. Significance criteria: ratio > 0.2 and FDR < 0.01. 

# Infer spatial cell-cell communication (~20 min)
adata = spacet.secact_spatial_ccc(
    adata,
    cell_type_col="cellType",
    scale_factor=1000,
    radius=20,
    ratio_cutoff=0.2,
    padj_cutoff=0.01,
)

# Access CCC results
ccc = adata.uns['spacet']['SecAct_output']['SecretedProteinCCC']
print(f"Significant CCC interactions: {len(ccc)}")
Output
Significant CCC interactions: 847

4. Visualize Cell-Cell Communication

The number in the heatmap represents the count of secreted proteins from senders to receivers. 

# CCC heatmap and circle plot
spacet.visualize_secact_heatmap(
    adata,
    colors_cell_type=cell_colors,
    row_sorted=True,
    column_sorted=True,
)

spacet.visualize_secact_circle(
    adata,
    colors_cell_type=cell_colors,
)
Heatmap of cell-cell communication. Rows represent sender cell types, columns represent receiver cell types. The number indicates the count of significant secreted protein interactions (padj < 0.01) between each pair.
Circle plot of cell-cell communication. Each node represents a cell type (colored by the legend). Directed edges indicate the number of secreted protein-mediated interactions from sender to receiver.
# Select specific cell types and proteins of interest
cell_types = ["Tumor_boundary", "Fibroblast", "Macrophage", "Endothelial"]

proteins = [
    "BGN", "COL1A1", "COL1A2", "DCN", "IGFBP5",
    "LGALS1", "LGALS9", "LYZ", "LUM", "MGP",
    "SPP1", "THBS1", "THBS2",
]

spacet.visualize_secact_dotplot(
    adata,
    sender=cell_types,
    secreted_protein=proteins,
    receiver=cell_types,
)
Dot plot of secreted protein-mediated cell-cell communication. Rows represent sender-protein combinations, columns represent receiver cell types. Dot size indicates the spatial co-localization ratio, and dot color indicates the significance (-log10 adjusted p-value).

5. Visualize Secreted Signaling Velocity

Select a significant CCC result and assign the sender, secreted protein, and receiver to visualize the signaling velocity. 

spacet.secact_signaling_velocity_scst(
    adata,
    sender="Fibroblast",
    secreted_protein="THBS2",
    receiver="Tumor_boundary",
    cell_type_col="cellType",
    show_coordinates=True,
    colors=cell_colors,
    point_size=0.1,
    legend_position="right",
    legend_size=2,
    arrow_color="#ff0099",
    arrow_size=0.2,
)
Full-tissue signaling velocity of THBS2 from Fibroblast (sender) to Tumor_boundary (receiver). Arrows indicate direction and magnitude of the signaling gradient. Cell types are color-coded by the legend.
